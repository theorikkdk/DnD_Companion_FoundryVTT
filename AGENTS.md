# Projet Foundry Companion

## Objectif
Créer une application web compagnon pour Foundry VTT, orientée joueurs D&D 5e.

## Contraintes produit
- Interface responsive téléphone et tablette
- Design moderne, sombre, ergonomique
- Navigation simple et tactile
- Application PWA installable
- Pour l’instant, utiliser uniquement des fausses données
- Ne pas intégrer Foundry dans cette première phase

## Contraintes techniques
- Utiliser React + Vite
- Utiliser TypeScript
- Utiliser Tailwind CSS
- Garder un code simple et compréhensible pour un débutant
- Préférer des composants réutilisables
- Éviter les dépendances inutiles

## Écrans à créer
- Écran de connexion (interface uniquement)
- Fiche personnage
- Actions
- Sorts
- Inventaire
- Lanceur de dés

## Données
Créer un personnage D&D 5e fictif stocké localement dans le projet.

## Style de travail
- Expliquer les changements importants en français simple
- Garder une structure claire
- Faire des modifications progressives
- Mettre à jour le README avec les instructions d’installation et de lancement
